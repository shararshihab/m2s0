# GitHub Pages Export

This folder contains the static portfolio version for GitHub Pages.

Upload these files/folders to a GitHub repository:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `data/`
- `.nojekyll`

The admin/backend files are intentionally not included because GitHub Pages cannot run Node.js, login sessions, uploads, SMTP, or API routes.

The portfolio loads content from `data/portfolio.json`.
